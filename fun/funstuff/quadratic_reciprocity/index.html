We first will review some definitions. If you are familiar, feel free to skip them!

Definition: Legendre Symbol $\Big(\frac{a}{p}\Big)$

Let $p$ be a prime number, and $a$ a natural number that $p$ does not divide. Then we can define: $$\bigg(\frac{a}{p}\bigg) = \left\{ \begin{array}{cc} 1 & \text{ if } \exists x\, (x^2 \equiv a \mod p) \\ -1 & \text{ if } \forall x\, (x^2 \not \equiv a \mod p) \end{array} \right\} $$

Definition: Permutation Sign $\Big[\frac{a}{p}\Big]$

Let $p$ be a prime number, and $a$ a natural number that $p$ does not divide. Then $a$ acts on $\mathbb Z /p \mathbb Z$ by sending $x \mapsto xa$. This map is has an inverse, since $a$ has an inverse modulo $p$. Therefore, $a$ is a permutation of $\mathbb Z /p \mathbb Z$. We can therefore, define the sign of the purmutation, which we denote $\Big[\frac{a}{p}\Big]$.
Recall any permutation can be uniquely (up to reordering) be decomposed into disjoint cycles. However, any permutation can be decomposed into a composition of transpositions. While two transposition decompositions may be different, there is always either an even or odd number of transpositions, regardless of the chosen decomposition. If there are an even number of transpositions, then the sign of the permutation is $1$. If there are an odd number of transpositions, then the sign of the permutation is $-1$.

Zolotarev's Lemma:
If $p$ is a prime number, and $a$ a natural number that $p$ does not divide, then: $$\bigg(\frac{a}{p}\bigg) = \bigg[\frac{a}{p}\bigg]$$.

Proof of Zolotarev's Lemma

Suppose that $(\frac{a}{p}) = 1$. Then there exists some $b$ coprime to $p$ such that $a \equiv b^2 \mod p$. Therefore, the action of $a$ on $\mathbb Z /p\mathbb Z$ is equivalent to the action of $b$ on $\mathbb Z/p\mathbb Z$ done twice. Since the sign of an action is a group homorphism, we see that $$\bigg[\frac{a}{p}\bigg] = \bigg[\frac{b}{p}\bigg] \cdot \bigg[\frac{b}{p}\bigg] = 1$$
Now we suppose that $(\frac{a}{p}) = -1$. First, we can see that $\text{ord}(a)$ in $(\mathbb Z/p\mathbb Z)^*$ is even. If this is the case, then $-1$ is a power of $a$.
Suppose for contradiction that $\text{ord}(a)$ is odd. Then there exists some $k\in \mathbb N$ such that $a^{2k+1} = 1$. Then $a^{2k+2} = a$, so $\big(a^{k+1}\big)^2 = a$, which is a contradiction.
Let the order of $a$ be $n$. When we decompose $a$ into disjoint cycles, each cycle has length $n$. The cycles are generated by the action of multiplication by $a$, so each cycle is exactly a coset of $(a)$. A cycle of even length has an odd sign. Therefore, the last thing we must check is there are an odd of number of cosets.
Suppose there are an even number of cosets. Since $(\mathbb Z/p\mathbb Z)^*$ is a cyclic group, then $(\mathbb Z/p\mathbb Z)^*/(a)$ is a cyclic subgroup. We see that there exists some nontrivial coset $b(a)$ such that $b^2(a) = (a)$. Therefore, $b^2 = a^k$ for some $k$. If $k$ is even, then $\pm b = a^{k/2}$. Since $-1$ is a power of $a$, then $b$ is a power of $a$. Therefore $k$ is odd. So there exists $t\in \mathbb N$ such that $2t+1 = k$. Finally, we can see that $(ba^{-t})^2 = a$, which is contradicts our assumption that there are an even number of cosets.
We have that action induced by $a$ is the composition of an odd number of odd cycles, so $[\frac{a}{p}] = -1$. Thus, we have proved Zolotarev's Lemma.
