Definition: Grothendieck Topology

A Grothendieck Topology on a category $\mathcal C$ is a collection of sets $\text{Cov}(X)$ called the covers of $x$ for every object $X$ in $\mathcal C$. The elements of $\text{Cov}(X)$ are called (unsurprisingly) covers of $X$. A cover of $X$ is a set of pairs $(Y,f)$ for some object $Y$ in $\mathcal C$ and some morphism $f:Y\rightarrow X$. I will often denote a cover as $\{f_i:Y_i\rightarrow X\}_{i\in I}$ for some index set $I$. We have the following requirements for this to be a Grothendieck Topology:

  1. If $f:Y \rightarrow X$ is an isomorphism then $\{f:Y \rightarrow X\} \in \text{Cov}(X)$.
  2. If $\{f_i:Y_i \> X\}_{i\in I} \in \text{Cov}(\mathcal C)$ and for each $i \in I$ we have an index set $J_i$ such that $\{g_j:Z_{i,j} \rightarrow Y_i\}_{j\in J_i} \in \text{Cov}(Y_i)$, then $\{(g_j \circ f_i):Z_{i,j}\rightarrow X\}_{i\in I,j \in J_i} \in \text{Cov}(X)$.
  3. If $\{f_i:Y_i \rightarrow X\}_{i\in I} \in \text{Cov}(X)$ and $g:Y \rightarrow X$ is a morphism of $\mathcal C$ then $Y_i\times_X Y$ exists for all $i\in I$ and $\{g_i: Y_i \times_X Y \rightarrow Y \}_{i\in I} \in \text{Cov}(Y)$ for the induced maps $g_i$.

Intuition: We require that an isomorphism is a cover, a cover of a cover is a cover, and the pullback of a cover is a cover.
We say a choice of category $\mathcal C$ and a topology $\tau$ on $\mathcal C$ is a site.

Definition: Sheaf

A sheaf of sets on a site $(\mathcal C,\tau)$ is a contravariant functor $\mathscr F: \mathcal C \rightarrow \text{Set}$ with the following property:
For any cover $\{f_i:Y_i\rightarrow X \}_{i\in I}\in \text{Cov}(X)$, the following diagram is an equalizer: $$\mathscr F(X) \xrightarrow{\mathscr F(f_i)}\prod_{i\in I} \mathscr F(Y_i) \rightrightarrows \prod_{j,k\in I}\mathscr F(Y_j \times_{X} Y_k) $$ In the case where $\mathscr F$ takes $\mathcal C$ to an abelian concrete category, such as $R-\text{Mod}$, the following sequence being exact is equivalent requirement: $$0\rightarrow \mathscr F(X) \xrightarrow{\mathscr F(f_i)}\prod_{i\in I} \mathscr F(Y_i) \xrightarrow{\mathscr F(p_j)-\mathscr F(p_k)} \prod_{j,k\in I}\mathscr F(Y_j \times_{X} Y_k)\rightarrow 0 $$

Now we construct a neat category, put two different topologies on the category, and construct a sheaf on both of them. It will turn out the gluing condition turns out to be exactly a well known theorem.
We construct the category $\mathbb N_{\text{div}}$ in following way. Let the objects be elements of $\mathbb N$ (I include 0, but it is not required). For any $a,b\in \mathbb N$, we construct a unique morphism from $a$ to $b$ if $a|b$. (This is poset category if you want to get technical.)
Next, we describe the first topology on $\mathbb N_{\text{div}}$. Notice that since there is at most one morphism between each object, a covering is dependent on the choice of objects. We say the collection $\{b_i\}_{i\in I}$ for some index set $I$ is a cover for $b$ if the following conditions hold:

  1. for all $i\in I$, $b_i|b$
  2. for all $i,j\in I$, $b_i$ and $b_j$ are coprime
  3. $b=\prod_{i\in I}b_i$ (here we are taking the product in $\mathbb N$, not the categorical product)

Note that the first condition implies the index set for any cover is finite, so the product in the third condition is well definied.

Lemma 1: This definition of a cover of $b$ defines a topology on all of $\mathbb N_\text{div}$.
